MessageBox Documentation

Introduction

MessageBox is a Unity GUI based implemenation of the .NET MessageBox class, however instead of inlining the results of the message box this system requires a callback to be written that will handle the result of the Message Box.

Installation

Installation is fairly easy and painless to do.

  1. Place the MessageBox prefab on a scene
  2. Set the don't destroy variable if you want the instance to persist between level loads.
  3. Set skin to a GUISkin if you don't want the default Unity skin to be used on the message boxes.
  4. Start using the MessageBox.Show() methods from your scripts.
